# Quiz Application

A Python-based Quiz Application designed to demonstrate core programming concepts through a simple and interactive quiz system. The application allows users to answer multiple-choice questions, calculate their score and percentage, and save the results for future reference.

## Features

* Start Quiz
* View Questions
* Answer Quiz
* Check Score
* View Percentage
* Save Results to a Text File
* Exit Application

## Technologies Used

* Python 3
* Functions
* Modules
* Lists
* Dictionaries
* Loops
* Conditional Statements
* File Handling
* Exception Handling

## Project Structure

* `main.py` – Main menu and program execution
* `questions.py` – Stores quiz questions
* `start_quiz.py` – Starts the quiz
* `view_questions.py` – Displays all questions
* `answer_quiz.py` – Processes user answers and updates the score
* `check_score.py` – Displays the current score
* `view_percentage.py` – Calculates and displays the percentage
* `save_result.py` – Saves quiz results to `result.txt`

## How to Run

1. Clone or download this repository.
2. Open the project in VS Code or any Python IDE.
3. Run `main.py`.
4. Select an option from the menu to use the application.

## Learning Outcomes

This project demonstrates modular programming, user input handling, quiz management, score calculation, percentage calculation, file operations, and error handling. It is suitable for Python beginners and academic mini-projects.
